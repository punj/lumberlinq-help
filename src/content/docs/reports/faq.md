## Reports

**Which report should I use for shipment performance?**

Use the Shipment Report. It includes shipment totals, CBM, pieces, trade type, parties, ports, BL number, and financial status.

**Which report should I use for receivables and payables?**

Use the Financial Report. It includes actual received, actual paid, receivable, payable, overdue, partner, currency, trend, and transaction views.

**Which report should I use for stock and processing?**

Use the Inventory Report. It shows available stock, in-process stock, shipment stock, movements, adjustments, and processing runs.

**Can I export reports?**

Yes. Supported reports show Excel and PDF export buttons.

**Why is a report empty?**

Check the date range and filters, then click Generate or Refresh.

## Manage Users

**How do I invite a user?**

Open Manage User, click Invite User, enter one or more email addresses, choose a role template if RBAC is enabled, and send the invite.

**Can I invite multiple users at once?**

Yes. Separate email addresses with commas, spaces, or new lines.

**What happens if an email is invalid?**

LumberLinq shows a validation message before the invite is sent.

**Where can I see pending invitations?**

Pending invitations appear below the user table on the Manage User page.

## RBAC

**What is RBAC?**

RBAC is Role-Based Access Control. It lets administrators control which modules and actions each user can access.

**What role templates are available?**

Field Operator, Supervisor, Analyst, Manager, and Custom.

**Can permissions be changed after inviting a user?**

Yes. Open the user Access Rights panel and adjust permissions at any time.

**Can I enable all permissions for one module?**

Yes. Each permission group supports Enable all and Disable all actions.

**What happens if RBAC is disabled?**

Users return to broader role-based access, while saved permission settings are preserved for later reactivation.
